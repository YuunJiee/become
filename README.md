# Become

個人技能追蹤工具，基於 100 小時法則 — 任何技能只要持續練習 100 小時，就能脫離初學階段。

## 功能

- 新增技能，選擇專屬顏色
- 每日記錄練習時間與備註
- 進度條追蹤當前階段進度
- 里程碑系統：Beginner → Practitioner (100h) → Skilled (300h) → Expert (1000h)
- 全年活動熱力圖
- 匯出 / 匯入 JSON 備份

## 使用

靜態網頁，直接開啟 `index.html` 或部署至 GitHub Pages。資料存於瀏覽器 localStorage，建議定期匯出備份。

## 技術

純 HTML / CSS / JS，無框架，無依賴。
